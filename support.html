Support

Spendly Log tracks your expenses entirely on your device — no account, no cloud sync, no network access.

Have a question, bug report, or feature request? Email mail@abhinav.se — I read every message myself. If Spendly Log saves you time, you can buy me a coffee ☕.

Frequently asked questions

Where is my data stored?

All expenses are stored locally on your device only, using Apple's on-device SwiftData storage. Nothing is uploaded anywhere. If you delete the app, your data is deleted with it — there is currently no backup or export feature beyond CSV import.

Does the app work without internet?

Yes — completely. Spendly Log has no networking code at all, so every feature (including screenshot and video import) works in airplane mode. Try it: turn on airplane mode and use the app; nothing changes.

How can I verify nothing is uploaded?

Three ways. First, the App Store listing shows Apple's privacy label as "Data Not Collected" — that declaration is part of App Review. Second, iOS itself keeps the receipts: enable Settings → Privacy & Security → App Privacy Report, use Spendly Log for a while, and check the Network Activity section — Spendly Log will show no network activity, ever. Third, everything works in airplane mode, which would be impossible if your data needed a server.

How does screenshot import work?

Take screenshots of the transaction list in your bank app, then in Spendly Log tap Import → Scan Screenshot and choose them (or copy a screenshot and tap Paste). Text recognition runs on your phone using Apple's Vision framework — the images are never uploaded. Every detected transaction appears in a review screen where you can fix categories or exclude rows before anything is saved.

How does video import work?

If the list is long, record your screen (Control Center → screen recording) while scrolling through it, then pick the video in the same import screen. Scroll slowly and steadily — fast flicks blur the text. Overlapping frames are de-duplicated automatically.

What does the orange dot on some imported expenses mean?

It marks a transaction whose date had to be guessed — usually because the screenshot cropped the list between a date heading and its rows. Open the expense, check or fix the date, and save; the dot disappears.

How do I delete an expense?

Swipe left on it in the list, long-press it for a menu, or open it and tap Delete Expense.

What CSV formats can I import?

The app auto-detects common bank and card export formats: comma, semicolon, or tab-delimited files; quoted fields; header detection; a single signed-amount column or split debit/credit columns; US (1,234.56) and EU (1 234,56) number formats; and several date formats, auto-disambiguated. If your bank's export doesn't import correctly, email a sample (redact the amounts if you like) and it can likely be supported.

Will importing the same file twice create duplicates?

No — every imported row is hashed, so re-importing a file you've already imported automatically skips rows that were previously added.

Can I edit the category CSV import assigns to a row?

Yes. The import preview screen lets you review every row before committing — toggle rows to exclude them, or tap a row's category to change it.

Does it support multiple currencies?

Amounts display using your device's locale currency. Multi-currency conversion isn't supported in this version.

Does it sync between my iPhone and iPad?

Not currently — each device keeps its own separate local data, since there's no cloud account to sync through.

Can I export or back up my data?

Yes — open the Import & Export menu and choose Export. You can share your expenses as a CSV or Excel spreadsheet, or a PDF statement grouped by month, for any period. Files are generated on your device and go only where you send them (Files, AirDrop, email, …). The CSV re-imports into Spendly Log without creating duplicates, so it doubles as a backup.
